# Worksheet Filler

Your personal AI worksheet tool — hosted on GitHub Pages.

## Deploy in 2 minutes

1. Go to github.com → Sign up for free account
2. Click **+** → **New repository**
3. Name it: `worksheet-filler`
4. Make it **Public**
5. Click **Create repository**
6. Click **uploading an existing file**
7. Drag BOTH files in (`index.html` and `README.md`)
8. Click **Commit changes**
9. Go to **Settings** → **Pages**
10. Under Source, select **main branch** → Save
11. Wait 1 minute → your site is live at:
    `https://YOUR_USERNAME.github.io/worksheet-filler`

## Your API key is safe
Your Claude API key is stored in YOUR browser only (localStorage).
It is never uploaded to GitHub or visible to anyone else.
